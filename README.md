🎭 Mystery Portfolio

A unique and interactive **Mystery Portfolio** built using **HTML, CSS, and JavaScript**. This project combines creative design, hidden clues, engaging animations, and interactive elements to create an intriguing user experience where visitors uncover information step by step.

## 🚀 Features

- 🕵️ Mystery-themed user interface
- 🎨 Responsive and modern design
- ✨ Smooth animations and transitions
- 🔍 Hidden clues and interactive elements
- 📱 Mobile-friendly layout
- ⚡ Built with pure HTML, CSS, and JavaScript
- 🎭 Creative storytelling approach

## 🛠️ Technologies Used

- **HTML5** – Structure and content
- **CSS3** – Styling, animations, and responsiveness
- **JavaScript (ES6)** – Interactivity and dynamic behavior

## 🎯 Purpose

The goal of this project is to showcase creativity and front-end development skills through a portfolio that feels more like a mystery adventure than a traditional website. Visitors explore hidden details, solve clues, and discover information about the creator in an engaging way.
